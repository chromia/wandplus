============
Installation
============

Requirements
============

* ImageMagick(verion 6 series, not 7)
* Wand(version ==0.4.4)

    see :doc:`Installation of wand <wand:guide/install>`


Install WandPlus
================

    .. code-block:: sh

        $ git clone https://github.com/chromia/wandplus.git
        $ cd wandplus/wandplus
        $ python setup.py install
